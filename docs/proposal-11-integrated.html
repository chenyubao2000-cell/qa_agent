Unified 8-Stage Responsibility Matrix

Stage Human (Decision) Agent (Intelligence) Automation (Execution)
1. Requirement Analysis Compose PRD, submit Issues, define SLA Parse PRD/Issue to extract test intent; identify uncovered scenarios from Langfuse traces git-watcher monitors PR and auto-triggers
2. Test Case Design Review key business cases (first time + spot checks) Generate cases using 6 design methods; classify by Tier 1-4; suggest eval rubrics —
3. Test Writing Review test suggestions in PR comments (one-click approve) unit-test-agent generates Vitest; e2e-orchestrator generates Playwright; eval-agent generates eval datasets —
4. Mock / Instrumentation Review mock accuracy (first-time config) mock-advisor analyzes external deps → generates config; LLM mock record/replay; StreamBridge specialized mock mock-server start/stop; MSW handler injection; MockLanguageModelV2
5. Test Execution — — Vitest/Playwright/k6 run in CI; Turborepo affected selective execution; sandbox isolation
6. Result Analysis Confirm bug classifications; review anomalous eval scores report-analyzer 3-class attribution; eval-agent trend analysis + regression detection; Flaky governance: stats → isolate → root cause Coverage report; eval scores to DB; flaky auto-isolation
7. Defect Location Determine fix priority Sentry stack trace → AST locate → correlate tests; failing test → code diff → suggest fix Sentry webhook trigger; source map parsing
8. Monitoring & Alerting Set thresholds and alert rules; decide rollback/continue sentinel-agent monitors all platforms; low eval score → root cause → suggest action; smoke fail → suggest rollback Sentry/Langfuse/Railway/DB webhooks and crons
