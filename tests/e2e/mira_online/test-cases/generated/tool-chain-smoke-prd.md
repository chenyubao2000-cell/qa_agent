# 多工具链冒烟测试 — 测试用例

## Method 1: Equivalence Partitioning

将多工具链场景的输入/输出按等价类划分：

**TC-PRD-TCS-001**: 多工具链任务正常完成并显示"任务已完成"
- **优先级:** P0
- **测试类型:** 等价类划分
- **前置条件:** 用户已登录，通过 fixture 创建包含多工具调用的任务（taskWithToolChainUrl）
- **操作步骤:** 1. 导航到 fixture 创建的任务页面 2. 等待任务完成
- **预期结果:** 对话区域显示"任务已完成"文本

**TC-PRD-TCS-002**: 多工具链任务生成的文件卡片可见（至少3种格式）
- **优先级:** P0
- **测试类型:** 等价类划分
- **前置条件:** 用户已登录，taskWithToolChainUrl fixture 任务已完成
- **操作步骤:** 1. 导航到 fixture 创建的任务页面 2. 等待任务完成 3. 检查对话区域的文件卡片
- **预期结果:** 对话区域至少显示3个不同格式的文件卡片（如 .md, .xlsx, .pptx, .pdf 等）

## Method 2: Boundary Value Analysis

**TC-PRD-TCS-003**: 多工具链任务生成的文件卡片包含预期文件类型标签
- **优先级:** P1
- **测试类型:** 边界值分析
- **前置条件:** taskWithToolChainUrl fixture 任务已完成
- **操作步骤:** 1. 导航到任务页面 2. 检查每个文件卡片的文件类型标签
- **预期结果:** 文件卡片显示的文件类型标签与实际扩展名一致（Excel、PPT、PDF 等）

## Method 3: Cause-Effect Graph / Decision Table

**TC-PRD-TCS-004**: 多工具链任务中工具调用卡片可见
- **优先级:** P1
- **测试类型:** 因果图
- **前置条件:** taskWithToolChainUrl fixture 任务已完成
- **操作步骤:** 1. 导航到任务页面 2. 检查对话区域是否存在工具调用卡片
- **预期结果:** 对话区域至少显示1个工具调用卡片（div[role="button"].rounded-xl.border.cursor-pointer）

## Method 4: State Transition Testing

**TC-PRD-TCS-005**: 点击 .xlsx 文件卡片打开预览面板
- **优先级:** P1
- **测试类型:** 状态迁移
- **前置条件:** taskWithToolChainUrl fixture 任务已完成，页面有 .xlsx 文件卡片
- **操作步骤:** 1. 导航到任务页面 2. 点击 .xlsx 文件卡片 3. 等待预览面板打开
- **预期结果:** 预览面板可见，面板标题包含文件名

**TC-PRD-TCS-006**: 点击 .pdf 文件卡片打开 PDF 预览
- **优先级:** P1
- **测试类型:** 状态迁移
- **前置条件:** taskWithToolChainUrl fixture 任务已完成，页面有 .pdf 文件卡片
- **操作步骤:** 1. 导航到任务页面 2. 点击 .pdf 文件卡片 3. 等待预览面板打开
- **预期结果:** PDF 预览面板可见，显示页码指示器（N / M 格式）

**TC-PRD-TCS-007**: 点击图片文件卡片打开图片预览
- **优先级:** P1
- **测试类型:** 状态迁移
- **前置条件:** taskWithToolChainUrl fixture 任务已完成，页面有图片文件卡片
- **操作步骤:** 1. 导航到任务页面 2. 点击图片文件卡片 3. 等待预览面板打开
- **预期结果:** 图片预览面板可见

**TC-PRD-TCS-008**: 预览面板关闭按钮可正常关闭
- **优先级:** P1
- **测试类型:** 状态迁移
- **前置条件:** 预览面板已打开
- **操作步骤:** 1. 打开任意文件预览 2. 点击关闭按钮
- **预期结果:** 预览面板关闭，文件名不再可见

## Method 5: Scenario Method

**TC-PRD-TCS-009**: "查看此任务中所有文件"按钮可见
- **优先级:** P1
- **测试类型:** 场景法
- **前置条件:** taskWithToolChainUrl fixture 任务已完成
- **操作步骤:** 1. 导航到任务页面 2. 滚动到文件区域 3. 检查"查看此任务中所有文件"按钮
- **预期结果:** "查看此任务中所有文件"按钮可见

**TC-PRD-TCS-010**: 文件卡片上下载按钮可见且可点击
- **优先级:** P1
- **测试类型:** 场景法
- **前置条件:** taskWithToolChainUrl fixture 任务已完成
- **操作步骤:** 1. 导航到任务页面 2. 找到第一个文件卡片 3. 检查下载按钮状态
- **预期结果:** 下载按钮可见且为启用状态

## Method 6: Error Guessing

**TC-PRD-TCS-011**: 点击 .pptx 文件卡片可正常打开预览或显示不支持提示
- **优先级:** P2
- **测试类型:** 错误猜测
- **前置条件:** taskWithToolChainUrl fixture 任务已完成，页面有 .pptx 文件卡片
- **操作步骤:** 1. 导航到任务页面 2. 点击 .pptx 文件卡片
- **预期结果:** 预览面板打开，显示幻灯片内容或"暂不支持"提示

**TC-PRD-TCS-012**: 点击下载按钮触发文件下载
- **优先级:** P1
- **测试类型:** 错误猜测
- **前置条件:** taskWithToolChainUrl fixture 任务已完成
- **操作步骤:** 1. 导航到任务页面 2. 点击第一个文件卡片打开预览 3. 点击预览面板中的下载按钮
- **预期结果:** 触发浏览器下载，下载文件名包含正确扩展名
