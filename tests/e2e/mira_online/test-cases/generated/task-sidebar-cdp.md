## 功能描述

Mira Task 页面侧边栏功能，包含：侧边栏折叠/展开、任务列表折叠/展开、任务右键菜单（分享、重命名、删除）及对应弹窗。

**页面 URL**: `https://www.mira.day/task`
**探查状态**: S0（初始）、S1（侧边栏折叠）、S2（任务列表折叠）、S3（右键菜单打开）、S8（分享弹窗）、S9（重命名弹窗）、S10（删除确认弹窗）

## Method 1: Equivalence Partitioning（等价类划分）

### 用户故事

**US-SIDEBAR-01**: 侧边栏可见性控制
作为 Mira 用户，我希望能折叠和展开侧边栏，以便获得更多主内容区域空间。

**US-SIDEBAR-02**: 任务列表区块折叠
作为 Mira 用户，我希望能独立折叠/展开任务列表区块，而不影响侧边栏整体状态。

**US-SIDEBAR-03**: 任务右键菜单
作为 Mira 用户，我希望通过任务的"更多"按钮打开右键菜单，对任务进行操作。

**US-SIDEBAR-04**: 分享任务
作为 Mira 用户，我希望通过分享弹窗创建分享链接，让他人查看会话。

**US-SIDEBAR-05**: 重命名任务
作为 Mira 用户，我希望通过重命名弹窗修改任务标题。

**US-SIDEBAR-06**: 删除任务
作为 Mira 用户，我希望通过删除确认弹窗安全地删除任务。

### 等价类分析

| 功能 | 有效等价类 | 无效等价类 |
|------|-----------|-----------|
| 侧边栏切换 | 点击 Toggle 按钮 | 无（无输入约束） |
| 任务列表折叠 | 点击"任务"区块标题 | 无 |
| 右键菜单 | 点击任务 More 按钮 | 无 |
| 重命名输入 | 非空字符串（≤maxLength） | 空字符串 |
| 删除确认 | 点击"删除"按钮 | 点击"取消"按钮 |

## Method 5: Scenario Method（场景法）

### 基本流 + 备选流

**备选流 1**: 打开右键菜单 → 按 ESC → 菜单关闭

**TC-CDP-SB-017**

- 优先级：P1

- 步骤：
  1. 点击"新建任务"按钮创建名为 "场景测试任务-{timestamp}" 的任务
